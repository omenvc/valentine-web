# Valentine's Day Website 💖

A fun and interactive Valentine's Day website built with React. Ask someone to be your Valentine with a playful twist!

## Features

- Beautiful pink gradient background
- Cute animated bear GIF
- Interactive buttons with fun behavior:
  - **Yes button**: Gets bigger each time the "No" button is avoided
  - **No button**: Runs away when you hover over it (you can't say no!)
- Celebration screen when "Yes" is clicked
- Fully responsive design

## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

The app will open in your browser at [http://localhost:3000](http://localhost:3000).

### Building for Production

```bash
npm run build
```

## How It Works

The "No" button moves to a random position whenever you try to hover over it, making it impossible to click. Meanwhile, the "Yes" button grows bigger each time, encouraging the user to click it instead!

## Technologies Used

- React 18
- CSS3 with animations and gradients
- Responsive design
